# 모델별 특징 요약

## 🤖 GPT-4o: 정밀한 지시 수행자
- **RLHF로 학습**되어 RAG 컨텍스트를 "지시사항"으로 해석하고 효과적으로 활용 (+8.33%)
- **가장 높은 정밀도** (29.41%)로 False Positive를 최소화하지만, Recall이 낮아 많은 알고리즘을 놓침
- **DH, DSA, RSA** 같은 전통적 공개키 암호에 강하지만, **한국 표준 암호**(16.2%)와 **SHA-2**(16.7%)에 매우 취약
- **구조화된 로그/설정 파일** 분석에서 44.74%로 최고 성능을 보이며 중간 속도(5.73초)
- 과도한 자신감(신뢰도 0.97)을 보이며 보수적 탐지 전략을 취함

## 🦙 Llama 3 8B: 독립적인 바이너리 전문가

- **가장 느린 속도**(13.57초)로 실시간 분석에 부적합하지만, RAG 없이 독립적으로 작동할 때 더 우수함
